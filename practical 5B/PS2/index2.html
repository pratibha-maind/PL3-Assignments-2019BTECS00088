Write a program in any language to find the maximum and minimum element in an array.

Implemenation code:


#include < stdio.h >

void findMinimumMaximum(int arr[], int N)
 {
   int i;
   int min = INT_MAX, max = INT_MIN;

   for (i = 0; i < N; i++)
   {
    if (arr[i] < min)
    {
     min = arr[i];
    }
    if (arr[i] > max)
    {
     max = arr[i];
    }
   }
   printf("The minimum element is %d", min);
   printf("\n");
   printf("The maximum element is %d", max);
   return;
  }
int main()
  {
   int arr[] = { 1, 2, 4, -1 };
   int N = sizeof(arr) / sizeof(arr[0]);
   findMinimumMaximum(arr, N)
;    return 0;
  }

Sample Input : 1, 2, 4, -1

Sample Output : -1, 4

Variables: N, i, min, max

Time complexity : O (N)
